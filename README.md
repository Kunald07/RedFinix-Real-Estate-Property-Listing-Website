# 🏠 RedFinix - Real Estate Website

RedFinix is a responsive and modern real estate website developed using HTML, Tailwind CSS, and JavaScript. The platform allows users to browse featured properties, view detailed information, and schedule property visits through an intuitive and visually appealing interface.

## 🚀 Features

* Responsive design for all devices
* Dynamic Hero Property Slider
* Smooth image and content transitions
* Property listing showcase
* Property details page
* Schedule property visit page
* Reusable Navbar and Footer components
* Modern UI built with Tailwind CSS
* Fast and lightweight front-end architecture

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* JavaScript (ES6)
* CSS3


## 🎯 Main Functionalities

### Hero Slider

* Displays featured luxury properties.
* Automatically rotates property images.
* Updates property title, description, price, and location dynamically.
* Interactive navigation dots for manual slide selection.

### Property Showcase

* Displays multiple property categories and featured listings.
* Clean card-based layout for property presentation.

### Property Details

* Dedicated page containing detailed property information.

### Schedule Visit

* Allows users to schedule property visits through a dedicated interface.

## 📱 Responsive Design

The website is fully responsive and optimized for:

* Mobile Phones
* Tablets
* Laptops
* Desktop Screens
